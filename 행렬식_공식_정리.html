행렬식 공식 정리

1. 2x2 행렬의 행렬식 정의:

\[A = \begin{bmatrix} a & b \\ c & d \end{bmatrix},\quad \det A = ad - bc\]

2. n × n 행렬식의 재귀적 정의:

\[\det A = \sum_{j=1}^{n} (-1)^{1+j} a_{1j} M_{1j}\]

3. 여인수(Cofactor)의 정의:

\[C_{ij} = (-1)^{i+j} M_{ij}\]

4. 라플라스 전개 정리 (i행 기준):

\[\det A = \sum_{j=1}^{n} a_{ij} C_{ij}\]

5. 라플라스 전개 정리 (j열 기준):

\[\det A = \sum_{i=1}^{n} a_{ij} C_{ij}\]

6. 삼각행렬의 행렬식:

\[\det A = a_{11}a_{22}\cdots a_{nn}\]

7. 행렬식의 기본 성질:

\[\begin{aligned} &\text{0행: } \det A = 0,\quad \text{교환: } \det B = -\det A,\\ &\text{동일행: } \det A = 0,\quad \text{스칼라곱: } \det B = k\det A,\\ &\text{선형결합: } \det B = \det A \end{aligned}\]

8. 기본 행렬의 행렬식:

\[\begin{aligned} &\text{두 행 교환: } \det E = -1,\\ &\text{한 행을 } k \text{배: } \det E = k,\\ &\text{한 행 더함: } \det E = 1 \end{aligned}\]

9. 행렬 곱과 행렬식:

\[\det(EB) = \det E \cdot \det B,\quad \det(AB) = \det A \cdot \det B\]

10. 가역성과 행렬식:

\[\det A \neq 0 \iff A \text{는 가역}\]

11. 스칼라 곱의 행렬식:

\[\det(kA) = k^n \cdot \det A\]

12. 역행렬의 행렬식:

\[\det(A^{-1}) = \frac{1}{\det A}\]

13. 전치행렬의 행렬식:

\[\det A = \det A^T\]

14. 크래머 법칙:

\[x_i = \frac{\det(A_i(\mathbf{b}))}{\det A}\]

15. 수반행렬 정의:

\[\text{adj}(A) = [C_{ij}]^T = [C_{ji}]\]

16. 역행렬과 수반행렬의 관계:

\[A^{-1} = \frac{1}{\det A} \cdot \text{adj}(A)\]
